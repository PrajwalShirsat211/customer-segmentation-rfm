# 🛒 Customer Segmentation using RFM + K-Means

## 📌 About
Segmenting customers based on purchasing behavior using 
RFM Analysis and K-Means Clustering.

## 🛠️ Tech Stack
- Python, pandas, numpy
- scikit-learn (K-Means)
- matplotlib, seaborn

## 📁 Files
- `customer_segmentation.ipynb` - Main notebook
- `rfm_segments.csv` - Output results

## 📊 Dataset
IBM Online Retail Dataset from Kaggle

## 🔁 How to Run
1. Download OnlineRetail.csv from Kaggle
2. Run the notebook cell by cell
